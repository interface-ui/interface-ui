# 悬浮操作按钮

悬浮操作按钮（FAB）帮助人们执行主要操作

在屏幕上使用悬浮操作按钮来执行最重要的操作。悬浮操作按钮会显示在屏幕上的所有其他内容之前，以其圆形形状和中央图标而闻名。

只使用悬浮操作按钮来呈现屏幕的主要操作。

悬浮操作按钮可以左对齐、居中或右对齐。它可以放置在导航栏之上，或者嵌套在导航栏中。
