# 单选框

单选框组允许用户从一组选项中选择一个选项。

当用户需要查看所有可用选项时，请使用单选按钮。如果可用选项可以折叠，考虑使用选择组件，因为它占用较少的空间。

单选按钮应默认选择最常用的选项。


## 用法

### 基本单选框

通常，`<Radio />` 与 `<FormLabel />` 组件一起使用，以显示标签。

使用 `v-model` 绑定变量和 `value` 作为选定的值

### 独立的单选按钮

`<Radio />` 也可以独立使用，无需 `<FormLabel />` 组件。
